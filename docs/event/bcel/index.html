What is this?

차분 제작자들 간의 교류 및 발전을 도모하고자 경쟁형 이벤트인 BMS chart Editors’ League를 개최합니다.
참가자는 난이도가 지정된 부문 혹은 자유 제작 부문에 참가하여, 공개된 과제곡의 차분을 작성합니다.
투고된 임프레를 기반으로 제출한 차분들의 순위를 겨룹니다.
이벤트는 2022년 6월 18일에 시작하여 약 10주 동안 BMS SEARCH venue에서 진행될 예정입니다.

譜面制作者たちの交流·発展を図るため、競争イベントBMS chart Editors' Leagueを開催します。
参加者はレベルが指定された部門、あるいは自由制作部門に参加し、公開された課題曲の差分を作成します。
作成した差分は投稿されたインプレに基づいて順位を競います。
イベントは2022年6月18日から、約10週間、BMS SEARCH venueで行われる予定です。

We hold a competitive event called "BMS Chart Editors' League" to promote communication and advance charting skills among chart makers.
Chart makers can participate in a category with a specified difficulty or a freestyle category to make the charts with given songs.
Submitted charts will be ranked based on the submitted impressions.
The event will start on June 18, 2022, and run for about ten weeks at BMS SEARCH Venue.

Rule

  1. 참가자는 신청서 작성시 참가 부문별로 추천곡 1곡을 작성합니다. 과제곡은 추천곡 리스트에서 추첨됩니다.
  2. 각 참가자는 7키 차분을 zip 파일 형식으로 마감일까지 등록해주세요.
    1. 참가한 부문의 과제곡 1곡 당 한 개의 엔트리를 등록해주세요.
    2. Freestyle 부문의 경우 모든 과제곡의 차분을 제출할 필요는 없습니다.
    3. 하나의 과제곡에 여러 개의 차분을 제출할 수 있으나, 평가를 받을 대표 차분 하나를 지정해주세요.
  3. 회장 등록 시 추정 난이도, 코멘트를 작성해주세요.
  4. 등록한 차분의 수정은 평가 기간 시작 전까지 가능합니다.
  5. 과제곡의 키음 어레인지를 허용합니다만, 회장 등록 시 키음 어레인지 사실을 코멘트에 작성해주세요.
  1. エントリーの際に参加部門につき1曲おすすめ必須です。課題曲はおすすめの曲リストから抽選されます。
  2. 7Key差分をzipファイルに圧縮して締め切りまでに提出してください。
    1. 参加した部門の課題曲1曲につき一つのエントリーで登録してください。
    2. Freestyle部門の場合、全ての課題曲の差分を提出する必要はありません。
    3. 一つの課題曲に複数の差分を提出できますが、インプレを受けられる代表差分一つを選んでください。
  3. エントリーのコメントに推定難易度を書いてください。
  4. 提出した差分の修正はインプレ期間の開始までできます。
  5. キー音アレンジもできますが、提出する際にキー音アレンジについてコメントに書いてください。
  1. Each participant must recommend a BMS for each category he has participated in. The songs will be selected from the list of recommended songs.
  2. Each participant should register a 7key chart in a zip file by the deadline.
    1. Please register one entry for each song in the category and round that you are participating in.
    2. A participant does not have to register a chart for all the songs for the freestyle category.
    3. A participant can register multiple charts for one song but must specify one representative chart to be evaluated.
  3. Please write down the estimated difficulty and comments when registering the venue.
  4. The registered charts can only be modified before the start of the evaluation period.
  5. Keysound (re)arrangements are allowed, but please write details about which (re)arrangements were made when registering to the venue.
