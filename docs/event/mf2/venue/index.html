Scoring System

2n명의 참가자로, n개의 팀이 참가할 때 최대득점은 아래와 같습니다.

2n名の参加者が参加し、n個のチームが構成される場合の最大得点は以下の通りです。

With 2n participants (n teams), the maximum available score for each team is as follows:

항목 1차 평가 1차 추론 2차 평가 2차 추론 최종 추론 총합
배점 5n 4n 5n 6n 3n 23n
項目 第1次評価 第1次推論 第2次評価 第2次推論 最終推論 合計
最大得点 5n 4n 5n 6n 3n 23n
Category Primary Evaluation Primary Inference Secondary Evaluation Secondary Inference Final Inference Sum
Score 5n 4n 5n 6n 3n 23n
  • 주최 팀의 추론 점수는 0점으로 고정됩니다.
  • 총 득점이 가장 높은 팀이 우승합니다.
  • 主催チームの推論スコアは0点に固定されます。
  • 最終的にスコアの合計が最も高いチームが優勝します。
  • The host team's inference score is fixed at 0 points.
  • The team with the highest total score will be the winner.
