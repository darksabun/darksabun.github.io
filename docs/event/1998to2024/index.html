What is this?

BMS chart Editor's Group에서 1998년, 1999년, ..., 2024년 각각 공개된 BMS 총 27곡으로 차분을 만드는 기획입니다.

BMS chart Editor's Groupで1998年,1999年,...,2024年それぞれで公開されたBMS計27曲で差分を作る企画です。

This is an event in which BMS chart Editor's Group creates charts with 27 BMS songs each from 1998 to 2024.

Rule

  1. 참가자는 1인당 년도를 최대 2개까지 선택할 수 있습니다. (1년에 1곡 선정)
  2. 7키 패턴을 하나 이상 포함해서 제출해주세요. (추가적인 5키, 9키, 10키, 14키 패턴은 재량 제출)
  3. 차분 제출 시 유효한 BMS 본체 링크와 곡과 패턴에 대한 코멘트를 txt 파일로 동봉해야 합니다.
  4. Anzu BMS Diff Tool 등을 이용해 차분의 엇갈림을 (즈레를) 체크해야 합니다.
  1. 担当は最高で1人2年まで(1年につき1曲)
  2. 7鍵譜面を必ず1つ以上提出してください。(5鍵,9鍵,10鍵,14鍵などは任意)
  3. 譜面を提出するとき、曲本体のリンク切れをチェックして本体のダウンロードリンクと自分が担当したレベル、譜面や曲に対する一言コメントをtxtファイルで添付してください。
  4. Anzu BMS Diff Toolなどを使って譜面のずれ、抜けをチェックしてください。
  1. Up to two songs per person per year are allowed. (Select one song per year)
  2. Please submit, including at least one 7key chart. (One could freely submit additional 5key, 9key, 10key, and 14key charts.)
  3. Please include a valid URL of BMS, and comment on patterns or songs in the text file.
  4. Verify charts do not have zure (misalignments) before submission.
