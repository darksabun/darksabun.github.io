What is this?

가면무도회 같은 차분 이벤트입니다.
각 참가자는 가명을 하나 정해서 1차 및 2차 등록에 차분을 1곡씩 제출합니다.
또한, 다른 가명을 사용하여 자유 부문에 차분을 여러 곡 제출할 수 있습니다.
이벤트에 등록된 가명의 정체를 가장 많이 맞추는 참가자가 우승합니다.

仮面舞踏会のような差分イベントです。
参加者は偽名を一つ選び、1次登録と2次登録でそれぞれ1曲ずつ差分を提出します。
また、自由部門には別の偽名を使って、複数の差分を提出できます。
登録された偽名が誰かを最も多く当てる参加者が優勝します。

This is a chart event, similar to a masquerade ball.
Each participant should submit a chart for primary and secondary registrations using the same alias.
Participants may also submit additional charts in the freestyle section using different aliases.
The participant who correctly guesses the most aliases will win.

Rule

  1. 각 참가자는 7키 차분을 zip 파일 형식으로 등록 기간에 가명으로 제출해주세요.
    1. 대표 가명을 정해서, 1차 및 2차 등록 기간에 대표 차분을 대표 가명으로 한 개씩 등록해주세요.
    2. 하나의 곡에 여러 개의 차분을 제출할 수 있으나, 한 엔트리에 동일한 명의로 등록해주세요.
    3. 대표 차분 이외의 곡으로 작성한 차분은 자유 부문에 제출 가능합니다. 대표 가명 이외의 가명을 사용해도 되며, 합작도 가능합니다. 단, 합작 시 엔트리 코멘트에 몇 명이 참여하였는지 명시해주세요.
  2. 회장 등록 시 추정 난이도, 참여인원(2인 이상인 경우), 코멘트를 작성해주세요.
    1. 코멘트에는 어떤 동봉패턴 기준으로 작성하였는지 명시해주세요. 또한, 키음 어레인지가 있는 경우 이를 명시해주세요.
  3. 등록한 차분의 수정은 평가 기간 시작 전까지 가능합니다.
  1. 7k差分をzipファイルで圧縮して登録期間に偽名で提出してください。
    1. 代表偽名を決めて、1次、2次登録期間に差分を代表偽名で1つずつ登録してください。
    2. 一曲に複数の差分を提出できますが、一つのエントリーに同じ偽名で提出してください。
    3. さらに、代表差分を作成した楽曲以外の曲で作った追加差分は自由部門に提出できます。代表偽名以外の偽名を使ってもいいですし、合作も可能です。合作する場合、エントリーのコメントで制作者の数を明かしてください。
  2. エントリーする時、推定難易度、制作者数(2人以上の場合)、コメントを記載いてください。
    1. コメントで何の同梱譜面を基準で差分を作成したかを明かしてください。キー音アレンジもできますが、コメントでキー音アレンジについて書いてください。
  3. 提出した差分の修正はインプレ期間が始まる前にできます。
  1. Each participant should register a 7key chart in a zip file under an alias.
    1. Please register one representative entry for each registration period under the same representative alias.
    2. One can register multiple charts for one song but must register in one entry.
    3. In addition to the representative charts, one can register additional charts with different songs on the freestyle section. Collaboration is allowed for the additional charts, but please specify how many participants joined in the entry.
  2. Please write down the estimated difficulty, the number of participants who joined (if two or more), and comments when registering the entry.
    1. Please identify the packaged chart on which the submitted chart is based in the comment. Keysound arrangements are permitted, but please list them in detail.
  3. The registered charts can only be modified before the start of the evaluation period.
