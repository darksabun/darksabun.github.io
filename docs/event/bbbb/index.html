What is this?

'BPM에 기반한 미니게임'을 주제로 하는 차분 이벤트입니다.
각 참가자는 악곡을 정해서 1차 및 2차 등록에 차분을 1곡씩 제출합니다.
제출한 악곡의 BPM이 참가자 평균에서 멀수록 높은 점수를 얻습니다.
단, 가장 높거나 낮은 BPM을 제출한 참가자는 예외로 점수를 얻지 못합니다.

この差分イベントは、「BPMをテーマにしたミニゲーム」として企画されています。
参加者は曲を選び、1ラウンドと2ラウンドでそれぞれ1曲ずつ差分を提出します。
提出された楽曲のBPMが参加者の平均値から離れるほど高得点を獲得できます。
ただし、最高BPMと最低BPMの楽曲を提出した参加者は得点対象外となります。

This is a chart event featuring a 'BPM-themed mini-game' with two rounds.
In each round, participants select a track for each round and submit a chart of it.
The further the BPM deviates from the participant's average, the higher the score one gets.
However, tracks with the highest or lowest BPM will not earn any points.

Rule

  1. 참가자는 각 라운드마다 1곡씩 선정하여 차분을 제출합니다.
  2. 7키 패턴을 하나 이상 포함해서 제출해주세요. (추가적인 5키, 9키, 10키, 14키 패턴은 재량 제출)
  3. 패턴을 제출할 때 유효한 BMS 본체 링크와 코멘트를 txt 파일을 첨부해야 합니다.
  4. Anzu BMS Diff Tool 등을 이용해 차분의 엇갈림을 (즈레를) 체크해야 합니다.
  1. 参加者は各ラウンドごとに1曲の差分を制作し、提出してください。
  2. 提出する際は、必ず7k譜面を1つ以上同梱してください。（5k、9k、10k、14k...などは任意です。）
  3. 本体のリンク切れがないか確認し、切れてないダウンロードリンクとコメントを同梱のtxtファイルに記載してください。
  4. Anzu BMS Diff Toolなどを使用し、譜面のズレや抜けを事前にチェックしてください。
  1. Each participant must submit one chart for a BMS track per round.
  2. Please ensure that at least one 7-key chart is submitted per round. (One could freely submit additional 5key, 9key, 10key, and 14key charts.)
  3. Additionally, prepare a text file containing a valid URL for the BMS and your comments.
  4. Before submission, please verify that your charts are free of any zure (misalignments).
