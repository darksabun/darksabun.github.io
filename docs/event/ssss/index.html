What is this?

각 참가자는 곡마다 요청사항이 있는 과제곡 3~5개의 목록을 제안합니다.
‘무작위로 선정된 다른 참가자’가 제안한 과제곡 목록에서 선곡하여, 요청사항에 맞는 차분을 제작합니다.
요청사항의 난이도가 높을수록, 임프레션의 평이 좋을수록 높은 점수를 얻습니다.
이벤트는 2024년 7월 1일에 시작하여 2개월간 BMS SEARCH venue에서 진행될 예정입니다.

各参加者は、各課題曲ごとにリクエストを付けた3～5曲のリストを提案します。
「ランダムに選ばれた参加者」が提案した課題曲リストから選曲し、リクエストに合った差分を作成します。
リクエストの難度が高いほど、インプレの評価が良いほど高い点数が得られます。
イベントは2024年7月1日から、約2ヶ月間、BMS SEARCH venueで行われる予定です。

Each participant proposes a list of 3 to 5 songs, with a specific request for each.
Participants then select one song from a list provided by 'a randomly selected participant' and create a chart that fulfills the request.
The chart's score increases with the difficulty of the request and the points of impressions.
The event will start on July 1, 2024, and run for about two months at BMS SEARCH Venue.

Rule

  1. 각 참가자는 7키 차분을 zip 파일 형식으로 등록 기간에 제출해주세요.
    1. (1차, 2차 등록) 할당받은 과제곡에서 1개를 선택하여, 요청사항에 맞는 차분을 제작해 제출해주세요.
    2. (자유 부문) 과제곡 목록에서 자유롭게 차분을 제작 및 제출해주세요.
    3. 하나의 곡에 여러 개의 차분을 제출하는 경우, 한 엔트리에 전부 등록해주세요.
  2. 회장 등록 시 추정 난이도, 코멘트를 작성해주세요.
    1. 코멘트에는 어떤 동봉패턴 기준으로 작성하였는지 명시해주세요. 또한, 키음 어레인지가 있는 경우 이를 명시해주세요.
  3. 등록한 차분의 수정은 평가 기간 시작 전까지 가능합니다.
  1. 7k差分をzipファイルで圧縮して登録期間に提出してください。
    1. （1回目、2回目の登録）振り分けられた課題曲から1つを選択し、リクエストに合った差分を製作して提出してください。
    2. （自由部門）課題曲リストから自由に選択して、差分を製作し、提出してください。 しかし、リクエストに合わない差分は、エントリーのコメントに明示してください。
    3. 1つの曲に多数の差分を提出することもできますが、1つのエントリーに全部登録してください。
  2. エントリーする時、推定難易度、コメントを記載してください。
    1. コメントで何の同梱譜面を基準で差分を作成したかを明かしてください。キー音アレンジもできますが、コメントでキー音アレンジについて書いてください。
  3. 提出した差分の修正はインプレ期間が始まる前にできます。
  1. Each participant should register a 7key chart in a zip file under an alias.
    1. (Primary & secondary registration) Please select one song from the received song list and submit a chart that meets the requested criteria.
    2. (Freestyle) Please freely submit charts from the list of proposed songs.
    3. If you submit multiple charts in one song, please register them all in one entry.
  2. Please write down the estimated difficulty, and comments when registering the entry.
    1. Please identify the packaged chart on which the submitted chart is based in the comment. Keysound arrangements are permitted, but please list them in detail.
  3. The registered charts can only be modified before the start of the evaluation period.

Impression Rule

조건

  • 플레이 리절트 URL을 코멘트에 남긴 모든 플레이어가 자유롭게 임프레션을 남길 수 있습니다.
  • 예외로, 이벤트 참가자는 플레이하지 않아도, 포인트와 레이블이 없는 코멘트를 남길 수 있습니다.

포인트 및 레이블

  • (1, 2라운드) 0-3 포인트
  • (자유 부문) 0-1 포인트
  • 레이블은 Request Complete! 또는 Request Incomplete? 중 하나를 선택합니다.
    1. 차분이 요청사항(조건)을 준수하고 곡과도 잘 어울린다고 느낀 경우, Request Complete!를 선택해주세요.
    2. Request Incomplete?를 선택한 경우, 선택 사유를 임프레션에 남겨주세요.
  • 플레이가 확인되지 않은 경우 포인트와 레이블은 무효입니다.

ルール

  • リザルト画像のURLを提出した全てのプレイヤーは、自由にコメントを残すことができます。
  • ただし、イベントの参加者は、プレイしなくてもポイントとラベルなしのコメントを残すことができます。

ポイントとラベル

  • （Round 1 & 2）0-3ポイント
  • （自由部門）0-1ポイント
  • ラベルはRequest Complete!またはRequest Incomplete?のどちらかを選択します。
    1. 差分がリクエスト（条件）に合って、曲ともよく合うと感じた場合は、Request Complete!を選択してください。
    2. Request Incomplete?を選択した場合、その理由をインプレに残してください。
  • プレイ記録が確認されないインプレのポイントとラベルは無効にします。

Rules

  • All players are free to leave comments as long as they include the URL of the result image at the top.
  • Event participants can leave comments without points and labels even if they haven't played.

Points & Label

  • (Round 1 & 2) 0-3 points
  • (Freeroll) 0-1 points
  • The label selects either Request Complete! or Request Incomplete?.
    1. If you feel that the chart meets the request and also fits well with the song, please be sure to select Request Complete!.
    2. If you select Request Incomplete?, please state the reason in the impressions.
  • Points & Label without confirmed plays are invalid.
