What is this?

BOFXVI에 제출된 작품으로 하루에 차분 1개를 작성해 업로드합니다.
11월 1일부터 11월 30일까지 30일간 진행되며, 주최자는 30개의 차분을 제출합니다.
게스트 참가 또한 가능합니다. BOFXVI에 제출된 작품의 차분을 투고하고 싶은 경우 Discord LuvTek#0302로 DM 바랍니다.

BOFXVIの作品で一日に差分1個を作成してアップロードしています。
11月1日から11月30日まで30日間行われます。
ゲスト参加もOKです。 BOFXVIの作品の差分を投稿したい場合Discord LuvTek#0302でDM願います。

M-SPIN writes and uploads one sabun per day with BOFXVI's songs.
It runs for 30 days from November 1st to November 30th.
We welcome the guest participation. If you want to upload your sabun for the project, please send DM via Discord LuvTek#0302.
