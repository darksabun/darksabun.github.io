What is this?

BMS chart Editor's Group에서 BOF21의 작품을 대상으로 작성한 차분을 모아 공개하는 기획입니다.
12월 8일까지 차분 제출을 받으며, 12월 14일 제출된 차분의 패키지와 난이도표를 공개합니다.

BMS chart Editor's GroupでBOF21の作品を対象に作成した差分を集めて公開する企画です。
12月8日まで差分の提出を受け、12月14日に提出された差分のパッケージと難易度表を公開します。

This is an event making the charts for BOF21's works by BMS chart Editor's Group
Submission is by 8th, December. The release of the package and difficulty table of submitted charts is on 14th, December.

Rule

  1. BOF21의 작품의 차분 보면을 제출해주세요.
  2. 패키지 공개 이전에도 업로더나 개인 사이트에서 자유로이 공개할 수 있습니다.
  3. 차분 제출 시 유효한 BMS 본체 링크와 추정 난이도를 txt 파일로 동봉해주세요.
  4. Anzu BMS Diff Tool 등을 이용해 차분의 엇갈림을 (즈레를) 체크해주세요.
  1. BOF21の作品の差分譜面を提出してください。
  2. パッケージを公開する前も、アップローダや個人サイトで自由に公開してもいいです。
  3. ダウンロードできる曲の本体リンクと推定難易度をtxtファイルで同梱して差分と一緒に提出してください。
  4. 譜面のズレ、抜けをチェックして提出してください(Anzu BMS Diff Toolなどのズレ、抜けを簡単にチェックできるツールもあります)。
  1. Please submit sabun chart(s) of BOF21's work. (Do NOT submit packaged charts)
  2. Before the package is released, you can freely upload the charts on the uploader or personal site.
  3. Please include a valid URL of BMS and the estimated difficulty of the charts in a txt file when submitting your chart(s).
  4. Verify charts do not have zure (misalignments) before submission.
