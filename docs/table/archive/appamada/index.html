몬스타씨의 요청에 따른 일부 패턴 제외에 대하여

이 아카이브는 WebArchive에 보관된 사본을 기반으로, 몬스타씨의 요청에 따라 일부 패턴을 제외하는 수정을 거쳤습니다.
드넓은 인터넷 공간을 유영하던 잔악무도한 WebArchive의 크롤러가, 몬스타씨의 미완성 난이도표를 GitHub Pages에서 발견, robots.txt가 없는 틈을 타, 몬스타씨의 의도에 반하여 무자비하게 난이도표를 아카이브하고 말았습니다.
DARKSABUN은 BMS 역사에 다시 없을 비극을 기록하고 널리 알리기 위해, 해당 아카이브를 재구성한 난이도표를 제공하고 있습니다.
미완성 난이도표의 원본은 DARKSABUN에서 제공하지 않지만, 여전히 비극의 현장서 열람 가능합니다.
