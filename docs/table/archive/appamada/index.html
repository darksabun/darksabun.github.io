알려드림

몬스타씨의 요청에 따른 일부 패턴 제외에 대하여

이 아카이브는 WebArchive에 보관된 사본 을 기반으로, 몬스타씨의 요청에 따라 일부 패턴을 제외하는 수정을 거쳤습니다.
해당 사본은 몬스타씨가 GitHub Pages에 게재한 미공개를 의도한 난이도표를, 수집 거부 처리가 이행되지 않은 때 아카이브 되었습니다.
현재 페이지에서 DARKSABUN 측이 제공하는 난이도표는, WebArchive 보관본 및 2024년 10월 22일 현재 몬스타씨가 제공하는 완성판과는 차이가 있으므로 유의 바랍니다.
